# Brick Breaker

A classic brick-breaker game built with HTML5 Canvas and vanilla JavaScript.

## How to Play

1. Open `index.html` in any modern browser
2. Click or press **Space** to start
3. Move the **mouse** to control the paddle (arrow keys / A/D also work)
4. Break all bricks to win!

## Features

- 5 rows of color-coded bricks with increasing point values
- Angle-based ball deflection off the paddle
- 3 lives per game
- Particle effects on brick destruction
- Ball trail effect
- Gradient bricks with highlight
- Start, Game Over, and Win screens

## Controls

| Input | Action |
|-------|--------|
| Mouse | Move paddle |
| Arrow Left / A | Move paddle left |
| Arrow Right / D | Move paddle right |
| Space / Enter / Click | Start / Restart |
